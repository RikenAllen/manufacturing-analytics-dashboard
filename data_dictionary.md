# Data Dictionary

Dataset: AI4I 2020 Predictive Maintenance Dataset

## Description

This synthetic dataset simulates industrial machine operating conditions and failure events. It will be used to analyze manufacturing KPIs, machine failures, process conditions, and maintenance-related trends.

## Columns

- UDI: Unique row identifier
- Product ID: Product identifier
- Type: Product quality/type category
- Air temperature [K]: Ambient air temperature in Kelvin
- Process temperature [K]: Process temperature in Kelvin
- Rotational speed [rpm]: Machine rotational speed
- Torque [Nm]: Torque applied by the machine
- Tool wear [min]: Tool wear duration in minutes
- Machine failure: Binary indicator for whether machine failure occurred
- TWF: Tool wear failure
- HDF: Heat dissipation failure
- PWF: Power failure
- OSF: Overstrain failure
- RNF: Random failure
