# Manufacturing Analytics Dashboard

This project analyzes manufacturing production and machine failure data to identify operational trends, calculate KPIs, and visualize insights through an interactive dashboard.

## Project Goals

- Clean and analyze manufacturing process data
- Calculate KPIs such as failure rate, downtime indicators, and machine performance trends
- Build an interactive dashboard using Python and Streamlit
- Present insights relevant to manufacturing operations and process improvement

## Tools Used

- Python
- Pandas
- Plotly
- Streamlit
- Scikit-learn
- Git/GitHub
